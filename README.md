# Water Quality Binary Classification Notebook
## Описание проекта
Классификация питьевой воды на основе набора данных water_potability.csv.  
Целью проекта является разработка предсказательной модели машинного обучения, которая на основе физико-химических параметров воды определяет, пригодна ли она для питья.  
Запуск app.py через команду python -m streamlit run app.py
